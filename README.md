# SC Bitching Betty

**Audible alert tool for Star Citizen** – monitors selected regions of your screen via OCR and colour recognition, and plays a configurable sound file whenever a threshold is met.

---

## Features

| Feature | Details |
|---------|---------|
| **Screen-region selector** | Click-and-drag overlay to pick any rectangle on any monitor |
| **OCR monitoring** | Reads text from the region and triggers on contains / exact / regex / numeric thresholds |
| **Colour monitoring** | Triggers when a configurable percentage of pixels match a target colour (within tolerance) |
| **Multiple monitors** | Add as many independent monitoring rules as you need |
| **Sound alerts** | Plays any `.wav` / `.ogg` audio file; per-monitor cooldown prevents spam |
| **Persistent config** | Settings saved to `config.json` and restored on next launch |

## Quick Start (Windows)

### 1 – Install Python
Download Python 3.11 or newer from <https://www.python.org/downloads/>.  
**Important:** tick *"Add Python to PATH"* during installation.

### 2 – Install dependencies
Double-click **`install.bat`** or run in a terminal:

```bat
install.bat
```

This installs the Python packages and generates a default `sounds\alert.wav`.

### 3 – Install Tesseract-OCR *(required for OCR monitors)*
Download the Windows installer from:  
<https://github.com/UB-Mannheim/tesseract/wiki>

After installing, open the app and set the Tesseract path in **File → Settings**.

### 4 – Run the application

```bat
python betty.py
```

## Usage

### Adding a monitor

1. Click **Add Monitor**.
2. Click **Select…** and drag a rectangle over the screen area you want to watch.
3. Choose **Trigger Type**:
   - **OCR (text)** – enter a *Trigger text* and *Match type*:
     - `contains` – fires if the text appears anywhere in the OCR result
     - `exact` – fires if the entire OCR result matches
     - `regex` – fires if a regular expression matches
     - `numeric_above` / `numeric_below` – fires when the largest number found by OCR crosses the threshold
   - **Colour** – pick a target colour, tolerance radius (in RGB Euclidean distance), and the pixel percentage that triggers the alert.
4. Browse for a **Sound File** (`.wav` / `.ogg`).
5. Set **Cooldown** (minimum seconds between repeated alerts) and **Poll interval**.
6. Click **OK**.

### Starting / Stopping
Click **▶ Start Monitoring** in the toolbar.  
The *Status* column updates live; rows flash ⚠ ALERT when triggered.  
Click **■ Stop Monitoring** to halt.

## Python dependencies

| Package | Purpose |
|---------|---------|
| `mss` | Fast cross-monitor screen capture |
| `Pillow` | Image processing |
| `pytesseract` | OCR wrapper for Tesseract |
| `numpy` | Colour distance calculations |
| `pygame` | Sound file playback |

Install with `pip install -r requirements.txt`.

---

## Troubleshooting

| Problem | Solution |
|---------|---------|
| *"Missing: mss, Pillow …"* in status bar | Run `pip install -r requirements.txt` |
| OCR always returns empty text | Install Tesseract-OCR and set its path in **File → Settings** |
| No sound plays | Ensure the sound file path is set correctly; install pygame if missing |
| Region selection window doesn't appear | Minimise other windows so the overlay can capture the screen |
